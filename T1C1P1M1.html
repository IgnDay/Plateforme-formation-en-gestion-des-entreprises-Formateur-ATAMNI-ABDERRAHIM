Notion de la Comptabilité Générale

📘 Définition

La comptabilité générale est une technique qui permet d’enregistrer toutes les opérations financières de l’entreprise durant une période appelée exercice comptable.

📊 Types de comptabilité

  • Comptabilité nationale : analyse de l’économie globale
  • Comptabilité publique : opérations de l’État
  • Comptabilité des entreprises :
    • Comptabilité générale
    • Comptabilité analytique
    • Comptabilité des sociétés

🎯 Rôle

  • Instrument d’information financière
  • Outil de gestion
  • Moyen de preuve juridique

💡 Utilité

  • Déterminer le résultat (bénéfice ou perte)
  • Aider à la prise de décision
  • Informer :
    • Dirigeants
    • État
    • Partenaires
    • Créanciers

⚖️ Caractère obligatoire

Toute entreprise doit tenir une comptabilité selon la loi 9-88 et le code de commerce.

Elle doit respecter le Plan Comptable Général pour garantir la normalisation.

📂 Sources

  • Cadre légal (loi 9-88, loi 44-03)
  • Code Général de Normalisation Comptable
  • Cadre comptable normal ou simplifié

📑 Obligations de l’entreprise

  • Établir et conserver les documents commerciaux
  • Établir et conserver les documents comptables
